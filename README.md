# LinkSage AI – Page-Level Internal Link Auditor

LinkSage AI is a Chrome extension that analyzes internal linking on any webpage in real time. It helps SEO professionals identify linking gaps, improve anchor text quality, and optimize pages for better search performance and AI extractability.

---

## 🚀 Features

* 🔍 Extract all internal links on a page
* 🧠 Classify anchor text (exact, partial, generic)
* 📊 Detect link placement (navigation, footer, content)
* ⚠️ Identify weak or missing internal links
* 🎯 Highlight High Priority Topic (HPT) linking gaps
* 📈 Provide actionable SEO recommendations

---

## 🧩 Use Cases

* Internal link audits at scale
* Content optimization workflows
* Improving topical authority
* Enhancing AI extractability signals
* Pre-publish SEO checks

---

## ⚙️ Installation

1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer Mode**
4. Click **Load Unpacked**
5. Select the extension folder

---

## 🛠️ How It Works

The extension injects a content script into the active page and extracts all anchor tags. It then filters internal links, analyzes anchor text, and categorizes links based on their placement in the DOM.
